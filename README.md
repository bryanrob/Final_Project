# Sixteen Stones - C# Edition
The game of Sixteen Stones in C#.

## Requirements
Dotnet 6.0 or later.

## User Instructions
Start with the following command:
```
dotnet run
```
The game will enter testing mode and begin using the *Test* class' *Main* function.
